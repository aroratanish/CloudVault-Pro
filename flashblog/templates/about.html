☁️ About CloudVault Pro

CloudVault Pro is a modern cloud storage and file management platform designed to help users securely organize, manage and share their files.

Built using Flask, SQLAlchemy and Bootstrap, CloudVault Pro provides a streamlined experience for file storage with powerful productivity features.


📁

Folder Management

Organize files into folders for a cleaner and more structured workspace.

🔗

File Sharing

Generate shareable links and collaborate with others easily.

🗑️

Trash Recovery

Restore accidentally deleted files from the trash bin instantly.


🚀 Key Features

  • Secure Authentication
  • Password Reset via Email
  • File Upload & Download
  • Folder Organization
  • Dashboard Analytics
  • Search Functionality
  • File Sharing Links
  • Trash & Restore System

🛠 Technology Stack

  • Python
  • Flask
  • SQLAlchemy
  • SQLite
  • Bootstrap
  • Flask-Login
  • Flask-Mail
  • Jinja2 Templates

CloudVault Pro was built as a full-stack project to explore authentication, database design, file management systems and cloud storage concepts.
